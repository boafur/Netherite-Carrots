# Netherite Carrots

Very simple mod that adds a few Netherite based items.


**Items:**

Netherite Nuggets:

- Used to craft Netherite Carrots
Â 

Netherite Carrots:

- Restores 8 hunger points
- Restores 16.6 saturation points
- Gives Regeneration 1
- Gives Fire Resistance
- Gives Resistance 2
- Gives Absorption 2

## Dependencies

FabricAPI > 0.32.5

## How to build it yourself

Clone the repository then run `.\gradlew build` in a terminal/command prompt window (if on linux then its `./gradlew build`)
